Política de Privacidad — AlbumPro 2026

Última actualización: Abril 2026

1. Información que recopilamos

AlbumPro 2026 recopila los siguientes datos para su funcionamiento:

  • Cuenta: nombre y correo electrónico mediante Google Sign-In o registro por correo.
  • Datos del álbum: las láminas que marcás como obtenidas o repetidas.
  • Grupos: nombre del grupo e integrantes con quienes compartís tu álbum.
  • Cámara: usada exclusivamente para escanear el dorso de las láminas mediante reconocimiento de texto (OCR). Las imágenes no se almacenan ni se envían a ningún servidor externo. El procesamiento ocurre completamente en el dispositivo.

2. Cómo usamos la información

Los datos recopilados se usan únicamente para:

  • Sincronizar tu álbum entre dispositivos.
  • Compartir el progreso con tu grupo familiar.
  • Facilitar intercambios de láminas repetidas.

No compartimos tu información con terceros ni la usamos con fines publicitarios propios.

3. Servicios de terceros

La aplicación utiliza los siguientes servicios de Google:

  • Firebase Authentication: para el inicio de sesión.
  • Cloud Firestore: para el almacenamiento de datos del álbum y grupos.
  • Firebase Cloud Messaging: para notificaciones push.
  • Google AdMob: para mostrar publicidad.

Estos servicios están sujetos a la Política de Privacidad de Google.

4. Retención y eliminación de datos

Podés eliminar tu cuenta y todos los datos asociados desde la sección de Perfil dentro de la aplicación. Tras la eliminación, los datos se borran permanentemente de nuestros servidores.

5. Seguridad

Los datos se transmiten mediante conexiones cifradas (HTTPS) y se almacenan en servidores seguros de Google Firebase.
